IMPORTANT DISCLAIMERS:

1. In Lies of P the point of contact with the enemy influences the amount of damage dealt (Usually +-10%). This means that while the base damage values will match these numbers, the actual in-game values will be different based on where you hit the enemy.

2. Lies of P uses a standard scaling number for each level of weapon reinforcement, which holds true for all but a small handful of weapons. As I do not know how to datamine the game for the new DLC reinforcement values, I am assuming they use the same reinforcement as in the base game. (Note, boss weapons all use the same reinforcement scalings.)

3. Reinforcement and player level modifiers are based on the values datamined by GrigLog on patch 1.5.0.0. I have no reason to believe these values have changed with the new patches, but if you notice scalings are inaccurate let me know and I'll look into it. As a rule, some of the values were +-5% when compared to the actual in-game values even within patch 1.5.0.0. (This game is weird.)
